API Reference

Specifications for methods and properties in the Keyboard.

Constructor

public Keyboard(editorElement: HTMLElement, isContentEditable: boolean, languageList: string[])

The Keyboard constructor. Keyboard will be added to the DOM when the constructor is called.

Call Keyboard.Show() to show the keyboard.

Parameters:

  • editorElement: The contenteditable div object to use as the editor.
  • isContentEditable: Whether the supplied element is contenteditable or not.
  • languageList: An array containing the lang property value(s) of the added keymap(s).

Static Methods


Keyboard.AddKeymap(keymap: object)

Adds the specified keymap object in the Keyboard.

See Docs for instructions on creating custom keymaps.

Once added, it can be used across all instances of the Keyboard.

Parameters:

  • keymap: The keymap object to add.



Keyboard.RemoveKeymap(name: string)

Removes the keymap whose name property matches the specified string.

Removing a keymap removes it from all instances of the Keyboard.

Parameters:

  • name: The name of the keymap object to remove.

Instance Methods


Keyboard.Show()

Shows the Keyboard. Override the .kb-shown CSS class to customize the animation.

Parameters: None



Keyboard.Hide()

Hides the Keyboard. Override the .kb-hidden CSS class to customize the animation.

Parameters: None



Keyboard.ToggleNumpad()

Toggles between numpad and character keys. The numpad keys are defined in the keys.numbers property of the keymap(s).

Parameters: None



Keyboard.SwitchCase()

Switches between lowercase and uppercase characters. The keys are defined in the keys.lowercase and keys.uppercase properties of the keymap(s).

Parameters: None



Keyboard.SetKeymap(name: string)

Sets the keymap as current whose name property matches the specified string.

Use the Keyboard.AddKeymap(keymap: object) static method to add new keymaps.

Parameters:

  • name: Name of the keymap to set as current.



Keyboard.SetLanguage(lang: string)

Sets the first keymap from added keymaps as current whose lang property matches the specified string.

Use the Keyboard.AddKeymap(keymap: object) static method to add new keymaps.

Parameters:

  • lang: Value of the lang property of the keymap to set.



Keyboard.SwitchLanguage()

Switches between languages defined in the languageList array while initializing the Keyboard.

Parameters: None
