Simple

Made with plain CSS styling using table display styles and viewport based sizing, the keyboard automatically adjusts to best view for both mobiles and tablets.

Light-weight

BackspaceJS has no dependencies. Made with TypeScript and minified using Gulp. The whole library weighs only ~7 kilobytes in production.

Customizable

You can customize the styling using custom styles in keymap, language and key mappings, fonts, and anything you want with custom keymaps.
