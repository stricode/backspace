Getting Started

A tutorial for setting up and running BackspaceJS quickly.

1. Get the library

Download the production-ready minified library from the GitHub repository (less than ~10KB)

https://github.com/stricode/backspace/

You can also download the TypeScript source file if you want.

2. Include the JS and CSS

Include the JS and CSS files in your app.


<link rel="stylesheet" href="css/backspace.min.css" />
<script src="js/backspace.min.js"></script>

3. Initialize the Keyboard

(i) Get a reference of your contenteditable div which you want to use as the editor.

var editor = document.getElementById('editor');

(ii) Declare an array containing values of the lang property of the keymap(s). The english language keymap is included in BackspaceJS by default.

var languages = ["english"];

The language switching key will appear if the array have more than one item.


(iii) Initialize a Keyboard instance.

var kb = new Keyboard(editor, true, languages);

The Keyboard constructor have the following parameters:

  • editorElement: A contenteditable div object to use as the editor.
  • isContentEditable: Whether the supplied element is contenteditable or not. Must be true.*
  • languageList: An array containing the lang property value of the added keymap(s).

Note: BackspaceJS does not yet support input elements. The isContentEditable parameter was added to support future versions.

4. Customize

Read the Docs and API reference for further instructions on styling, creating custom keymaps and more.

Contributions to the library are also welcome. You can report issues and create pull requests on GitHub.
